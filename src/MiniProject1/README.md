# Mini Project 1 – Exploratory Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on a customer analytics dataset to understand data structure, clean inconsistencies, visualize patterns, and extract insights.

## Project Structure
MiniProject1/
│
├── data/
│   ├── customer_analytics.csv
│   └── customer_analytics_cleaned.csv
│
├── notebooks/
│   └── MiniProject1_EDA.ipynb
│
├── reports/
│   └── MiniProject1_Documentation.pdf
│
├── README.md
└── requirements.txt

## Steps Performed
1. Data Inspection
2. Data Cleaning
3. Univariate & Bivariate Analysis
4. Correlation Analysis
5. Insights & Executive Summary

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
